A class type safe wrapper for a ConfigurableFactory that takes a desired target class and returns null if the factory returns an incompatible object. This allows the called to cast the return values without worrying about ClassCastExceptions.

static ConfigurableFactory games.stendhal.server.core.config.factory.ConfigurableFactoryHelper.getFactory ( final String  className)
static

Create an object factory using a [logical] class name.

This will attempt to create a factory for the class in the following order:

  • If a class named <class-name&gtFactory exists and implements ConfigurableFactory, return an instance of it.

  • If a class named <class-name&gt exists and implements ConfigurableFactory, return an instance of it.

  • If a class named <class-name&gt exists and accepts a constructor with ConfigurableFactoryContext, return a factory that creates an instance with that constructor when used.

  • If a class named <class-name&gt exists and has a default constructor, return a factory that creates an instance with that constructor when used.

  • Returns null,
Parameters
classNameA base class name to load.
Returns
A factory, or null if no valid class was found.
